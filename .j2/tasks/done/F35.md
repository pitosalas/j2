# F35 — Install Script Adopt Message

## Tasks

### T01 — Detect existing source files in target directory
**Status**: done
In `scaffold/install.sh`, after the "Installation complete" message, add a check: if the target directory contains any `.py`, `.js`, `.ts`, `.rs`, `.go`, `.java`, `.cpp`, or `.c` files (searched recursively, excluding `.j2/` and `scaffold/`), set a flag indicating this is an existing project.

### T02 — Print adopt hint when existing code detected
**Status**: done
When the existing-project flag is set, append an additional line to the "Next steps" output: `"  5. Run /adopt in Claude Code to scan your existing codebase and generate a spec and feature list."`. When no existing code is detected, the output remains unchanged.

### T03 — Write tests for adopt message
**Status**: done
Add a test that verifies: (1) `install.sh` prints the adopt hint when source files exist in the target, (2) `install.sh` does not print the adopt hint for an empty target directory. Use a temp directory with a dummy `.py` file for the positive case.
