# Project Specification: Hello World

## Overview
A minimal command-line program that prints "Hello, World!" to stdout.

## Users
Developers learning the j2 framework workflow. This spec exists to test the simplest possible end-to-end project.

## Problem
Demonstrates that the j2 workflow functions correctly with a trivially simple project.

## Key Features
- Print "Hello, World!" to stdout when run
- run with uv run helloworld

## Constraints
- Python 3.10+

## Out of Scope
- Command-line arguments
- Localization
- Any output other than "Hello, World!"
